# schoonhoven.com

Personal website built with React and TypeScript, deployed to GitHub Pages.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint
```

## Deployment

The site is automatically deployed to GitHub Pages on every push to the `main` branch using GitHub Actions.

Custom domain: [schoonhoven.com](https://schoonhoven.com)
